    Create api/schemas.py with five Pydantic v2 BaseModel classes. No Config or model_config
    needed — defaults suffice. Keep descriptions concise since these appear in /docs.

    Imports: from pydantic import BaseModel, Field; from typing import List.

    PredictRequest:
      text: str = Field(..., min_length=1, max_length=2000, description="Full Vietnamese review text")
      target: str = Field(..., min_length=1, max_length=200, description="Target phrase — must be exact substring of text")

    AspectProb (shared by PredictResponse.aspect_probs and sentiment_probs):
      label: str    — aspect label string (e.g. "ROOMS#CLEANLINESS") or sentiment string (e.g. "POSITIVE")
      score: float  — softmax probability rounded to 4 decimal places

    PredictResponse:
      aspect: str                     — top predicted aspect label
      sentiment: str                  — top predicted sentiment; one of POSITIVE, NEGATIVE, NEUTRAL
      aspect_probs: List[AspectProb]  — top-5 aspect predictions sorted descending by score
      sentiment_probs: List[AspectProb] — all 3 sentiment scores sorted descending by score
      latency_ms: int                 — inference wall time in milliseconds

    HealthResponse:
      status: str       — always the string "ok"
      model_loaded: bool
      device: str       — "cpu", "cuda", or "" if not yet loaded

    ModelsResponse:
      active: str        — encoder key string e.g. "phobert_large" (per D-05)
      encoder: str       — resolved HuggingFace model name e.g. "vinai/phobert-large"
      attention_key: str — e.g. "target_conditioned_attention"
      label_structure: str — e.g. "multitask_aspect_sentiment"
  </action>
