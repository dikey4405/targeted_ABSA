    Create api/main.py. Structure the file in this order to prevent issues with route/middleware
    registration order:

    1. IMPORTS (all at top, no inline imports):
       Standard library: os, from contextlib import asynccontextmanager, from pathlib import Path.
       FastAPI: FastAPI, HTTPException, from fastapi.middleware.cors import CORSMiddleware,
       from fastapi.responses import JSONResponse.
       Local: from api.schemas import PredictRequest, PredictResponse, HealthResponse, ModelsResponse;
       from api.inference import InferenceEngine.

    2. MODULE-LEVEL SINGLETON:
       _engine: "InferenceEngine | None" = None
       Use a string annotation (forward reference style) to avoid type-checker issues on
       Python < 3.10.

    3. LIFESPAN CONTEXT MANAGER (per API-06 — single load at startup):
       @asynccontextmanager
       async def lifespan(app: FastAPI):
           global _engine
           _engine = InferenceEngine()   # reads all config from env vars internally
           yield
           _engine = None

    4. FASTAPI APP INSTANCE:
       app = FastAPI(title="Targeted ABSA Inference API", version="1.0.0", lifespan=lifespan)

    5. CORS MIDDLEWARE (per API-04 — add BEFORE route definitions):
       Read _cors_raw = os.getenv("CORS_ORIGINS", "http://localhost:5173,http://127.0.0.1:5173")
       _cors_origins = [o.strip() for o in _cors_raw.split(",") if o.strip()]
       app.add_middleware(
           CORSMiddleware,
           allow_origins=_cors_origins,
           allow_credentials=False,          # CRITICAL: must be False with explicit origin list;
                                             # setting True with a non-wildcard list causes
                                             # browser to block preflight on some agents
           allow_methods=["GET", "POST"],
           allow_headers=["Content-Type", "Accept"],
       )

    6. GET /health ENDPOINT (per API-02):
       @app.get("/health", response_model=HealthResponse)
       def health():
           if _engine is None or not _engine.ready:
               return JSONResponse(
                   status_code=503,
                   content={"status": "ok", "model_loaded": False, "device": ""},
               )
           return HealthResponse(status="ok", model_loaded=True, device=str(_engine.device))

    7. GET /models ENDPOINT (per API-03):
       @app.get("/models", response_model=ModelsResponse)
       def models():
           if _engine is None:
               raise HTTPException(status_code=503, detail="Model not ready")
           return ModelsResponse(
               active=_engine.encoder_key,
               encoder=_engine.encoder_name,
               attention_key=_engine.attention_key,
               label_structure=_engine.label_structure,
           )

    8. POST /predict ENDPOINT (per API-01, API-05, API-06):
       @app.post("/predict", response_model=PredictResponse)
       async def predict(req: PredictRequest):
           # Pydantic has already validated min_length=1 on req.text and req.target.
           # Additional NFC normalization and substring check happen inside engine.predict().
           if _engine is None or not _engine.ready:
               raise HTTPException(status_code=503, detail="Model not ready")
           async with _engine.lock:
               try:
                   result = _engine.predict(req.text, req.target)
               except ValueError as exc:
                   raise HTTPException(status_code=422, detail=str(exc)) from exc
               except Exception as exc:
                   raise HTTPException(status_code=500, detail=str(exc)) from exc
           return PredictResponse(**result)

    9. STATIC FILE GUARD (future Phase 2/3 integration, zero impact in Phase 1):
       After ALL route definitions, add a conditional block that mounts the compiled React
       frontend if frontend/dist exists. This block must come LAST because FastAPI evaluates
       routes top-down; a wildcard catch-all registered before /predict would intercept API calls.
       Do NOT import StaticFiles or FileResponse unconditionally — guard the import inside the if block.
       if Path("frontend/dist").exists():
           from fastapi.staticfiles import StaticFiles
           from fastapi.responses import FileResponse
           _frontend_dist = Path("frontend/dist")
           app.mount("/assets", StaticFiles(directory=str(_frontend_dist / "assets")), name="assets")
           @app.get("/{full_path:path}")
           def serve_spa(full_path: str):
               idx = _frontend_dist / "index.html"
               if idx.exists():
                   return FileResponse(str(idx))
               raise HTTPException(status_code=404)
       In Phase 1 this block is unreachable. Phase 3 builds frontend/dist and it activates.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → FastAPI /predict | req.text and req.target are fully untrusted user-supplied strings |
| FastAPI → InferenceEngine | Pydantic-validated input forwarded to ML model; all exceptions wrapped before HTTP response |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-01 | Spoofing | All endpoints — no authentication | low | accept | Per SPEC §Boundaries: public demo, no auth in scope. All network-accessible requests accepted. Acceptable for localhost-only demo use case. |
| T-03-02 | Tampering | CORS allow_credentials | medium | mitigate | allow_credentials=False enforced in CORSMiddleware. Explicit origin list (not "*") prevents unintended cross-origin access. Combination of allow_credentials=True + non-wildcard origins is used in the wrong direction by some CORS clients — setting False is the safe default for a credential-free public API. |
| T-03-03 | Information Disclosure | HTTP 500 response body | medium | mitigate | All exceptions in /predict are caught; only str(exc) (the message string) is returned as the detail field — no Python traceback, no internal file path, no stack frame information exposed. |
| T-03-04 | Denial of Service | asyncio.Lock request queuing | low | accept | Single asyncio.Lock serializes /predict requests indefinitely (no timeout). Queue grows under concurrent load. Explicitly accepted: single-user demo, low traffic, no SLA. asyncio.Lock is a deliberate design per API-06. |
| T-03-05 | Tampering | Input text (crafted Unicode / long payloads) | low | mitigate | NFC normalization in InferenceEngine.predict() (Plan 02). Pydantic Field max_length=2000 on text (Plan 01) bounds tokenization memory. Text is passed directly to HuggingFace tokenizer with truncation=True — oversized inputs are truncated, not rejected, which is safe. |
| T-03-SC | Tampering | Makefile pip install commands | low | accept | Packages confirmed legitimate by deployment.md §3 research. No [ASSUMED] or [SUS] packages. The Makefile install target is a developer-run convenience; it does not execute during plan execution by the agent. |
</threat_model>
